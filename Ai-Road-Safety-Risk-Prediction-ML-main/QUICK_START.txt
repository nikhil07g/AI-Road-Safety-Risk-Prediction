## 🔧 IF SOMETHING GOES WRONG

### Issue: "No module named 'X'"
```bash
pip install torch torchvision opencv-python numpy pillow
```

### Issue: "Data not found"
Verify dataset path in script:
```python
'data_root': r'c:\Users\D Sai tejashwini\Downloads\Telegram Desktop\archive (1)\cityscapes_data\cityscapes_data\processed'
```

### Issue: Very slow training
This is normal on CPU! Can reduce batch size:
```python
config['batch_size'] = 16  # Reduce memory usage
```

### Issue: Out of memory
```python
config['batch_size'] = 8   # Or smaller
```
