## 🚀 Ready for Training

### Normalization Parameters
Use these values to normalize images during training:

```python
# BGR Format (OpenCV/Torch)
MEAN = [85.31, 77.16, 86.89]
STD = [52.25, 46.7, 53.5]

# Normalization formula:
# normalized = (image - MEAN) / STD

# For torch.transforms.Normalize (requires 0-1 range):
# MEAN = [85.31/255, 77.16/255, 86.89/255] = [0.334, 0.303, 0.341]
# STD = [52.25/255, 46.7/255, 53.5/255] = [0.205, 0.183, 0.210]
```

### Step 1: Copy Dataset
```bash
# Copy processed dataset to your training machine
```

### Step 2: Set Up Training Environment
```bash
pip install torch torchvision
pip install opencv-python numpy pillow
# Install your model library (e.g., segmentation_models_pytorch)
```

### Step 3: Load Data
```python
from cityscapes_dataloader import CityscapesDataManager

dm = CityscapesDataManager(
    data_root='path/to/processed',
    batch_size=32,
    num_workers=4
)

train_loader, val_loader = dm.get_train_val_loaders()
```

### Step 4: Train Model
```python
for epoch in range(epochs):
    for batch_images in train_loader:
        # Forward pass
        outputs = model(batch_images)
        loss = criterion(outputs, targets)
        
        # Backward pass
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
    
    # Validation
    with torch.no_grad():
        for val_images in val_loader:
            val_outputs = model(val_images)
            val_loss = criterion(val_outputs, val_targets)
```
