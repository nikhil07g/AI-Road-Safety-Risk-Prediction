### Data Augmentation (Already Implemented)

```
Training Only (with probabilities):
  • Horizontal flip: 50%
  • Vertical flip: 30%
  • Rotation: ±15 degrees
  • Brightness: ±20%
  • Contrast: ±10%
  • Gaussian blur: 20%
```

### Hyperparameter Tuning Guide
```
If loss decreases slowly:
  → Increase learning rate (1e-3 → 5e-3)
  → Increase batch size (32 → 64)
  → Reduce weight decay (1e-4 → 1e-5)

If loss oscillates/diverges:
  → Decrease learning rate (1e-3 → 5e-4)
  → Reduce batch size (32 → 16)
  → Increase weight decay (1e-4 → 1e-3)

If model overfits:
  → More aggressive augmentation
  → Increase weight decay
  → Reduce model capacity
```

### Monitoring During Training
```
✓ Training loss should decrease
✓ Validation loss should decrease (but may oscillate)
✓ Learning rate should decrease over time
✓ No NaN or Inf values in loss
✓ Checkpoints saved automatically
```

## 💾 USING TRAINED MODELS

### Loading Best Model for Inference
```python
import torch
from train_segmentation_model import SimpleSegmentationModel

# Load model
device = 'cuda' if torch.cuda.is_available() else 'cpu'
model = SimpleSegmentationModel(num_classes=19)
checkpoint = torch.load('best_model_epoch_X.pth', map_location=device)
model.load_state_dict(checkpoint['model_state_dict'])
model.to(device)
model.eval()

# Inference on image
from cityscapes_dataloader import CityscapesDataset
dataset = CityscapesDataset('./processed/val')
image = dataset[0]  # Single image

with torch.no_grad():
    output = model(image.unsqueeze(0).to(device))
    prediction = output.argmax(dim=1)  # Class index per pixel

print(prediction.shape)  # [1, 256, 512]
```

### Saving for Deployment
```python
# Save model weights only (smaller file)
torch.save(model.state_dict(), 'model_weights.pth')

# Load for inference
model = SimpleSegmentationModel(num_classes=19)
model.load_state_dict(torch.load('model_weights.pth', map_location='cpu'))
```
